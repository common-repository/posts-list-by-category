With this plugin list of posts from each category can be displayed. 

== Description ==
This plugin gives you the flexibility to display all your wordpress posts in a categorical manner under a single post or a page.From the plugin configuration page you can also list your category and posts as per the order you want.

**Features** 

* Easy to use and require only a single tag to start it working
* Works both with the wordpress < 2.3 and > 2.3 database structure
* Category and posts ordering configuration feature

== Installation ==

1. Download "Posts list by Category" plugin.
2. Open /wp-content/plugins folder at your web server.
3. Upload the folder "posts-list-by-category" there.
4. Goto Plugins page in your wordpress back office.
5. Activate "posts-list-by-category" plugin.
6. Go to Settings >> Posts list by Category.
7.Add the following tag <!-- postslistbycategory --> in html source box to make it active
Note: This plugin will require one time free registration.
